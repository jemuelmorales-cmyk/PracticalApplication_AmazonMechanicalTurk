# PracticalApplication_AmazonMechanicalTurk
Driving scenarios and Coupons Acceptance

This project is an Exploratory Data Analysis. It investigates what makes drivers accept or reject coupons. It has demographics of the driver including, sex, age, marital status, have childern, education level and occupation. Included in the data is the weather, time of day and if they have passengers.There are 5 coupon types: cheap restaurants, coffee houses, carryout, bars, and pricier restaurants.
